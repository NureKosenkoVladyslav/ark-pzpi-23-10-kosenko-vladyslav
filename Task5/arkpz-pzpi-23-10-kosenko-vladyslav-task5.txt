1. Текстовий опис інженерних рішень
Програмна система «MelodyTrack» розроблена як екосистема для цифровізації музичної освіти. Основні інженерні рішення включають:
1.	Централізована обробка: Дані від ІоТ-датчиків обробляються в хмарі, забезпечуючи високу доступність (99,9%) та безпеку.
2.	Технологічний стек: Система побудована на мові C#, фреймворку .NET 8, веб-сервері ASP.NET Core Web API та базі даних MSSQL.
3.	Інтелектуальний ІоТ-клієнт: Реалізовано Smart-пристрій, який виконує локальну математичну обробку даних (Edge Computing) перед відправкою на сервер.
2. Реалізація функціональності Vision & Scope
Згідно з розділом 2.2 документа «Vision and Scope», було реалізовано наступну функціональність:
1.	MF-1 (Електронний журнал): Автоматичне логування оцінок на основі даних репетицій.
2.	MF-2 (Керування репертуаром): Прив'язка оцінок до конкретних предметів (наприклад, Фортепіано).
3.	Бізнес-логіка сервера: Реалізовано метод GetAverageGrade для аналітичної звітності (MF-3).
